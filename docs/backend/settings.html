Plugin Settings

On Backend, goto Settings, CMS, Magic Forms.

Hide navigation item

It's possible to hide top menu item.

This is useful if you are planning to use events with your custom plugin.

reCAPTCHA

Configure reCAPTCHA parameters.

GDPR

If your site is under European GDPR law, you can enable auto delete of records after "x days".

For this setting to work, you need to be using Task scheduling from OctoberCMS.

Note: configuring task scheduling is beyond the scope of this plugin/documentation.
